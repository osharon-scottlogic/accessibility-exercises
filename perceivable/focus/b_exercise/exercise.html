Visual focus

We were tasked to provide a contact form. The previous developer, whose name will not be mentioned wrote the code of the contact form below. Have a look at the contact form and click on the fields. As you can see the form fields are not highlighted when you click on them. They lack visual focus.

  • Fix the visual-focus issues in the form.
  • Ensure the first field has default focus.

Keyboard focus

Try to navigate through it using your keyboard. You can use the following controls:

    • Tab: Move to the next interactive element
    • Shift + Tab:Move to the previous interactive element
    • Arrow keys: Navigate through radio buttons and checkboxes
    • Space: Activate the currently focused element
    • Enter: Submit the form
  • Ensure the form so that it can be navigated using only a keyboard
Extensions

Use a screen reader

Now that your form has the correct visual and keyboard focus, you can also have a listen to what this form sounds like using a screen reader. To activate and deactivate the narrator on windows, use Ctrl + Windows key + Enter

  • Navigate the form using a screen reader. Can you navigate the form with your eyes closed?
  • What do you notice? Make a note of anything that could be improved.
  • Try improving the form for screen reader use.
